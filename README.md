SmartAppointment is a simple appointment booking system built with ASP.NET Core MVC.
It allows users to register, log in, view available consultants, and book appointments based on availability.
The project demonstrates role-based access control, authentication with cookies, and real-world database relationships.

🚀 Features
🔐 User Registration and Login with Cookie Authentication
👤 Role-based Authorization (User, Admin, Consultant)
📅 Appointment Creation and Cancellation
📆 Consultant Availability Management
🧭 Admin Access to Manage Users and View All Appointments
📊 Entity Framework Core (EF Core) for database access

🛠️ Tech Stack
ASP.NET Core MVC (.NET 8)
Entity Framework Core (Code First)
SQL Server
Bootstrap (for UI)
Cookie-based Authentication
